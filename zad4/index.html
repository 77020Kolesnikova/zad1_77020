O mnie

Jestem początkującym programistą z solidnymi podstawami w językach JavaScript, Java, Kotlin, Python oraz C#. Posiadam praktyczne umiejętności w tworzeniu stron internetowych (HTML/CSS) oraz pracy z relacyjnymi bazami danych, takimi jak SQL Server i Microsoft Access. Moje wykształcenie techniczne uzupełniam biegłą znajomością systemów operacyjnych Windows oraz zaawansowaną obsługą narzędzi pakietu Microsoft Office.

Wyróżnia mnie wysoki poziom kompetencji językowych (polski C1, angielski B2), co pozwala mi na swobodną pracę w środowisku międzynarodowym oraz korzystanie z dokumentacji technicznej. Jestem osobą odpowiedzialną, dobrze zorganizowana i stale rozwijam swoje umiejętności poprzez samodzielną realizację projektów programistycznych. Moim celem jest dalszy rozwój zawodowy w branży IT oraz wnoszenie wkładu w pracę zespołową.
